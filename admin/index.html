Import Shopee CSV

From Shopee Affiliate → Product Offer → select items → Batch Get Link → download CSV. Batch caps at 100 rows, so select all your CSVs at once (ctrl/cmd-click) to merge them. No images in that export — items import with a blank thumbnail; edit them in later.

Import Shopee Images

1. Open Shopee Affiliate in Chrome — browse/search products until cards are visible
2. Click the FragLy Extractor extension icon → EXTRACT PRODUCTS → COPY JSON
3. Paste JSON below → PREVIEW → APPLY
Copy Missing List → paste once into the extension's "Missing List" box. It'll then only surface products FragLy actually needs while you browse Shopee — no eyeballing which ones already have photos.
